NFT Generator

Python tool that generates rare NFT collections using layered image logic and metadata automation.

Behind the Scenes

The NFT generator was built using Python and the Pillow library to programmatically combine image layers (like backgrounds, characters, accessories) into thousands of unique outputs.

It also generates metadata in JSON format for each NFT to be compatible with platforms like OpenSea. The system includes rarity weighting, batch preview generation, and export tools.
